What is TLS?

  • Transport Layer Security
  • A cryptographic protocol
  • Replaces SSL (Secure Sockets Layer)

What does TLS do?

TLS helps with:

  • Confidentiality
  • Integrity
  • Authenticity

Remember: CIA

Key Concept: Man in the Middle

Anything between the two endpoints trying to communicate, such as:

  • Routers
  • Wi-Fi access point
  • DNS servers
  • Wire (cat 5, copper, fibre optic, etc.)
  • Many, many, many, many other things...

Confidentiality

  • Communication is encrypted
    • Stops man in the middle from snooping
  • Uses symmetric encryption
  • Keys used are uniquely generated for each connection

Integrity

  • Communication cannot be tampered with
    • Stops man in the middle from altering content
  • Uses symmetric encryption
  • Keys used are uniquely generated for each connection

Authenticity

  • Verifies the server's identity
    • Stops impersonation by a man in the middle
  • Uses public-key encryption
  • Typically relies on delegated authority

TLS Handshake

  • Determines cipher suite to use
  • Authenticates the server (optionally, the client as well)
  • Establishes and shares session key information

Key Concept: Public-key vs Symmetric Encryption

  • Public-key encryption
    • Leverages public information
    • Computationally costly
  • Symmetric encryption
    • Requires a shared secret
    • Computationally cheaper than public-key
